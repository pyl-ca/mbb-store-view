# 商城管理系统

基于 Vue 3 + TypeScript + Element Plus 的现代化电商管理平台

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📋 功能特性

- ✅ **商品管理**: 完整的商品CRUD、分类管理、SKU管理
- ✅ **评论系统**: 评论展示、发布、管理、审核
- ✅ **退款系统**: 退款申请、审核、资金管理
- ✅ **用户管理**: 用户信息、地址、收藏管理
- ✅ **订单管理**: 订单查看、状态跟踪
- ✅ **后台管理**: 统计分析、权限控制

## 🔧 技术栈

- **前端框架**: Vue 3 + TypeScript
- **UI组件库**: Element Plus
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **HTTP客户端**: Axios
- **构建工具**: Vite
